Detailed Description

This is a library for the HDC1010 I2C Temperature & Humidity Sensor

Designed specifically to work with the Oak Development Technologies HDC1010 Acorn -—> urlhere

The sensor uses I2C to communicate with additional pins that can be connected to for other operational modes.
